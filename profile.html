🤖 Groq API Key（AI 解讀功能）

填入你的 Groq API Key，即可在幣種技術診斷頁面使用 AI 解讀功能。
Key 僅儲存在你的帳號中，不會被其他人看到。
👉 點此申請免費 Groq API Key

1. 前往 https://console.groq.com 用 Google 帳號註冊
2. 右上角點 API Keys → Create API Key
3. 填名稱，選 No expiration，勾選驗證後送出
4. 複製 Key（格式：gsk_開頭）貼到下方儲存
